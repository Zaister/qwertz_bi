QWERTZ Shortcuts for Bigger Interface
=====================================

This mod is an add-on for the "Bigger Interface" mod and replaces all "z" 
shortcuts with "y", so all the shortcuts are nicely arranged in the bottom
row as intended on a German QWERTZ-style keyboard. Ironman compatible.

Dieser Mod ist eine Erweiterung zum Mod "Bigger Interface" und ersetzt alle
Tastaturkürzel "z" durch "y", so dass dann alle Kürzel wie vorgesehen schön
auf der untersten Reihe einer deutschen QWERTZ-Tastatur zu finden sind.
Kompatibel mit Ironman-Modus.
